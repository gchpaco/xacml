A Brief Tour of DrScheme version 209

209
Released December 2004

DrScheme is an interactive, integrated programming environment designed specifically with the needs of beginners in mind. This tour highlights some of the features that make it especially useful in education.

DrScheme asks you at startup what language level you want. If you're using DrScheme as part of a course, your instructional staff should be able to tell you what language level to use. If you're a beginner learning Scheme, we strongly encourage you to use Beginning Student under the How to Design Programs tab under Teaching Languages. If you're already familiar with Scheme, use either Standard (R5RS) if you want just the language standard, or Pretty Big under the PLT tab for a richer language. You can always change the language level using the Language | Choose Language... menu item.

This tour is part of an extensive help system called Help Desk. The tour includes references to other portions of Help Desk that you can consult for additional information on a topic. You can re-visit this tour through the front page of Help Desk.
