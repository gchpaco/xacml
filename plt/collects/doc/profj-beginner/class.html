class

  • class NAME { MEMBER MEMBER ... }
    This form creates a class with the given name that extends Object. One member must be a constructor.
  • class NAME extends NAME { MEMBER MEMBER ... }
    This form creates a class with the given name that inherits from the class named by extends. One member must be a constructor.
  • abstract class NAME { MEMBER ... }
    This form creates an abstract class with the given name that extends Object. Members can include abstract methods.
  • abstract class NAME extends NAME { MEMBER ... }
    This form creates an abstract class with the given name that inherits from the class named by extends. Members can include abstract methods.
