ProfessorJ Beginner Language

  • PROGRAM
    • IMPORT ... DEF ...
  • IMPORT
    • import QUALIFIED_NAME ;
    • import QUALIFIED_NAME .*;
  • DEF
    • CLASS-DEF
  • CLASS-DEF
    • class NAME { MEMBER MEMBER ... }
    • class NAME extends NAME { MEMBER MEMBER ... }
    • abstract class NAME { MEMBER ... }
    • abstract class NAME extends NAME { MEMBER ... }
  • MEMBER
    • CONSTRUCTOR
    • METHOD
    • FIELD
  • CONSTRUCTOR
    • NAME (TYPE NAME, ...) { super(EXPRESSION , ...); ASSIGNMENT ...}
    • NAME (TYPE NAME, ...) { ASSIGNMENT ...}
  • METHOD
    • TYPE NAME (TYPE NAME, ...) { STATEMENT }
    • abstract TYPE NAME (TYPE NAME, ...);
  • FIELD
    • TYPE NAME;
  • ASSIGNMENT
    • this.NAME = EXPRESSION;
  • STATEMENT
    • if (EXPRESSION) STATEMENT else STATEMENT
    • return EXPRESSION;
  • EXPRESSION
    • EXPRESSION OP EXPRESSION
    • -EXPRESSION
    • EXPRESSION && EXPRESSION
    • EXPRESSION || EXPRESSION
    • !EXPRESSION
    • this
    • EXPRESSION.NAME( EXPRESSION ...)
    • super.NAME( EXPRESSION ...)
    • EXPRESSION.NAME
    • new NAME( EXPRESSION ...)
    • (EXPRESSION)
    • NAME

    • CHARACTER

    • false
    • STRING
  • QUALIFIED-NAME
    • NAME. ...NAME
  • NAME
    • a sequence of keyboard characters not including certain characters and not forbidden to use as a name
  • TYPE
    • prim-type
    • NAME
  • OP
