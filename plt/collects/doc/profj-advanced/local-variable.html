local-variable

  • TYPE DECL-INIT , DECL-INIT ...;
    This form creates a variable with the given name and type which can be used within the block declaring it, after the declaration. If the statement is present outside of a block, the name is valid within the method body after its declaration.
  • TYPE DECL-INIT , DECL-INIT ...;
    This form creates a variable with the given name and type which can be used within the expression and statements of the for loop.
