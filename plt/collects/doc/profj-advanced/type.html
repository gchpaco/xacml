type

  • TYPE
    This will specify the type of the value the method will return, and indicates the method will return a value
  • prim-type
    Prim-types include boolean, char, byte, short, int, long, float, and double
  • NAME
    Class types are specified by the name of the class
  • TYPE[]
    This type indicates an array of TYPE, where it is possible to have an array of arrays
