name

a sequence of keyboard characters not including certain characters and not forbidden to use as a name
Names may not contain any of these characters: space ~ ! @ # % ^ & * ( ) - + = { } [ ] | \ : ; " ' > < , . ? / Or be any of the following words abstract, boolean, break, byte, case, catch, char, class, const, continue, default, do, double, else, extends, final, finally, float, for, goto, if, implements, import, instanceof, int, interface, long, native, new, package, private, protected, public, return, short, static, strictfp, super, switch, synchronized, this, throw, throws, transient, try, void, volatile, while.
